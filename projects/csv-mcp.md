Rather than web site building, I branched into something different. I've been wanting to understand model context protocol and what better way than trying to create a MCP server that I can use in Claude Desktop.

Fairly simple server that monitors a folder of .csv files and exposes both those files in a simplified form as resources and a range of tools to work with them.

Useful project as a learning exercise and can actually see myself myself making use of the MCP server.
