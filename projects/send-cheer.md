I took the idea of the one off Happy Birthday site from April and converted it into a Send Cheer, a platform for sharing celebrations for a range of events.

Currently fully functional, but I have a few ideas for additional features and needs some tidy up work too.

I like the fact that each celebration gets its own OG image so if shared via Messages or most social media there is a large image and some branding.
